Operadoes lógicos

"x = y" Atribuir valores

"x == y" Igual, compara se X e Y são iguais (ignora o tipo da variavel)

"x === y" identico, verifica se X e Y são identicos

"x !== y" não identico, verifica se X e Y não são identicos

"x != y" diferente, verifica se X e Y são diferentes

"x < y" menor, verifica se X é /menor que Y

"x <= y" menor ou igual, verifica se X é menor ou igual a Y

"x > y" maior, verifica se X é maior que Y

"x >= y" maior ou igual, verifica se X é maior ou igual a Y
